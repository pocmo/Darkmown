# Darkmown

A native macOS Markdown viewer with syntax highlighting, Mermaid diagrams, dark mode, table of contents, in-page search, HTML export, and print support.

## Features

- **Syntax Highlighting** — Code blocks rendered with highlight.js, with theme-matched light/dark styles
- **Mermaid Diagrams** — Render flowcharts, sequence diagrams, and more directly in your Markdown
- **Dark Mode** — Automatic, light, or dark appearance with smooth transitions
- **Table of Contents** — Auto-generated sidebar from headings with click-to-scroll navigation
- **In-Page Search** — Find text with match highlighting and previous/next navigation
- **HTML Export** — Export your rendered document as a standalone HTML file
- **Print Support** — Print documents with clean, print-optimized styles
- **Zoom Controls** — Zoom in, out, and reset for comfortable reading
- **Status Bar** — Word count, character count, and estimated reading time
- **Drag & Drop** — Open Markdown files by dropping them onto the window
- **GFM Support** — Full GitHub Flavored Markdown including tables, task lists, and line breaks

## Requirements

- macOS 13.0+
- Xcode 15+

## Getting Started

1. Clone the repository
2. Open `Darkmown.xcodeproj` in Xcode
3. Build and run

## Keyboard Shortcuts

| Shortcut | Action |
|---|---|
| `Cmd + O` | Open file |
| `Cmd + F` | Find in document |
| `Cmd + G` | Next match |
| `Cmd + Shift + G` | Previous match |
| `Cmd + Shift + S` | Toggle table of contents |
| `Cmd + Shift + E` | Export HTML |
| `Cmd + P` | Print |
| `Cmd + +` | Zoom in |
| `Cmd + -` | Zoom out |
| `Cmd + 0` | Reset zoom |
